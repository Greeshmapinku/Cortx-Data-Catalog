--------------------------------------------------------------------------------
STEP 1: SETUP (2 minutes)
--------------------------------------------------------------------------------

1. Open PowerShell in this folder

2. Create virtual environment:
   python -m venv venv

3. Activate it:
   venv\Scripts\activate

4. Install dependencies:
   pip install -e ".[dev]"

--------------------------------------------------------------------------------
STEP 2: RUN DEMO (2 minutes)
--------------------------------------------------------------------------------

1. Set the API key:
   $env:GROQ_API_KEY="your_groq_api_key_here"

2. Run the demo:
   python run_demo.py

   OR:
   
   cortx-catalog-gen --demo

3. Check the outputs:
   - catalog.json (full metadata)
   - tool_manifest.json (MCP manifests)
   - demo_data/ (synthetic test data)

--------------------------------------------------------------------------------
STEP 3: VERIFY TESTS PASS (1 minute)
--------------------------------------------------------------------------------

   pytest

--------------------------------------------------------------------------------
STEP 4: DEPLOY LIVE DEMO (For GitHub/Submission)
--------------------------------------------------------------------------------

OPTION A: Render (Recommended - Free Hosting)
1. Push this folder to GitHub
2. Go to render.com → Sign up with GitHub
3. New Web Service → Select this repo
4. Set Environment Variable:
   GROQ_API_KEY = your_groq_api_key_here
5. Deploy! You'll get a URL like:
   https://cortx-catalog-demo.onrender.com

OPTION B: Local Testing
   python app.py
   (Then open http://localhost:5000 in browser)
